# README

How to set up

1. Install Ruby version 3.2.0
2. Install rails version 8.0.1

Check versions:
ruby -v
Output is something like this: ruby 3.2.0 (2022-12-25 revision a528908271) [x86_64-linux]

rails -v
Output: Rails 8.0.1

After ruby and rails is installed, 
Install gems:
bundle install


Launch app:
rails s
